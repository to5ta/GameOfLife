# GameOfLife

Application to simulate [Conways Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) using Python 3.10 with PySimpleGUI 4.57.0


## GUI
* Start/Stop Simulation
* Record frames on/off
